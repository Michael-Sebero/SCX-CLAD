## **I.M.P.E.R.A.T.O.R**

**Integrated - Multitiered - Priority - Execution - Ranked - Adaptive - Topology - Ordered - Runtime**

The goal of this scheduler is to improve CPU related bottlenecks and framerate consistency. 

### **How to Build and Install:**

Enter the commands below into the console.

```
git clone https://github.com/Michael-Sebero/SCX-IMPERATOR && cd /home/$USER/SCX-IMPERATOR && cargo build --release 

sudo mv /home/$USER/SCX-IMPERATOR/target/release/scx_imperator /usr//bin/ && chmod 777 /usr//bin/scx_imperator

scx_imperator
```
